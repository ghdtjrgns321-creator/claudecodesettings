# Skills (목차)

전역 스킬 = 언어·도메인 무관한 작업 방식·도구. **핵심 하네스는 superpowers 플러그인이 제공**, 도메인/스택 스킬은 각 프로젝트로 이동.
스킬은 각 `SKILL.md`의 `description`을 Claude가 보고 자동 노출한다(별도 배선 파일 없음).

> 최종 정리: 2026-07 (40 → 12). 삭제·이동·링크정리 내역은 `../전역설정-체계.md` 참조.

| TDD·디버깅·검증·서브에이전트·brainstorming·스킬작성               | **superpowers 플러그인**(`superpowers:*`)                |

## 운영 규칙 (목차 동기화)

- 스킬 추가·삭제·이동 시 **이 목차를 같은 작업에서 갱신**한다.
- 새 스킬을 전역에 두기 전, superpowers·프로젝트 스킬과 역할이 겹치는지 확인 — 겹치면 전역에 두지 않는다.
- 전역은 "언어·도메인 무관"만. 스택·도메인 지식은 프로젝트로.
