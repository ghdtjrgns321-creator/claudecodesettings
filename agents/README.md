# Agents (목차)

전역 에이전트 = 여러 단계를 자율 수행하는 부하 일꾼. **겹치는 역할은 위임이 불안정(flaky)해지므로 최소로 유지.**
나머지 역할은 빌트인·superpowers 플러그인이 커버한다.

> 최종 정리: 2026-07-05 (6 → 2). 삭제분은 `backups/pre-agent-cleanup-2026-07-05.tar`.

## 유지 에이전트 (2)

| 에이전트                    | 역할                                      | 언제                               | 실사용     |
| --------------------------- | ----------------------------------------- | ---------------------------------- | ---------- |
| **code-reviewer**           | 한국어 코드 리뷰(품질·보안·성능·아키텍처) | 기능/버그/리팩터 완료 후 자동 실행 | 21회(주력) |
| **documentation-architect** | 개발 문서 작성·검증·다이어그램            | 문서 작성/대량 문서 수정 시        | 3회        |

## 다른 역할은 이걸로 (전역 에이전트 안 만듦)

| 역할              | 대체 수단                                             |
| ----------------- | ----------------------------------------------------- |
| 계획 수립         | 빌트인 `Plan` 에이전트 · `superpowers:writing-plans`  |
| 탐색·리서치       | 빌트인 `Explore` · `deep-research` 스킬 · `WebSearch` |
| 디버깅            | `superpowers:systematic-debugging`                    |
| 리팩토링          | 빌트인 `simplify` · `superpowers`                     |
| 보안 검토         | 빌트인 `/security-review` 커맨드                      |
| 서브에이전트 실행 | `superpowers:subagent-driven-development`             |

## 운영 규칙 (목차 동기화)

- 에이전트 추가·삭제 시 **이 표를 같은 작업에서 갱신**한다.
- 새 에이전트를 만들기 전에 위 "대체 수단"과 역할이 겹치는지 확인 — 겹치면 만들지 않는다.
