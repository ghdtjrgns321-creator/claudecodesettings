# 02. CLAUDE.md 구성 방식

전역 규칙 파일. **43줄 / ~1.1k 토큰** (권장 5k 이하). 매 세션 로딩되므로 판단 규칙만 담고 상세는 스킬·훅으로 위임.

## 섹션 구성

| 섹션      | 내용                                                      |
| --------- | --------------------------------------------------------- |
| 정체성    | 사용자 = AI-Native 바이브 코더                            |
| 소통      | 한국어·결론 먼저·대명사 금지·쉬운 말                      |
| 착수      | 이해+가정 2줄 확인 · 요청한 것만 · 교정 2회면 세션 재시작 |
| 전수·다건 | census 3단계(분모 고정→흔적→차집합) · `미검증` 라벨       |
| 코딩      | 간결·근본원인·리터럴 금지·한글 파일 주의                  |
| Git·안전  | 커밋 규칙·AI 문구 금지·시크릿                             |

## 여기 넣는 것 / 넣지 않는 것

| 판정                                  | 위치                        |
| ------------------------------------- | --------------------------- |
| 매 턴 참인 **상수** 원칙              | CLAUDE.md                   |
| 기계로 검사 가능한 강제               | 훅                          |
| 가끔 쓰는 절차                        | 스킬                        |
| 스스로 발동 조건을 판정해야 하는 규칙 | 넣지 않음 (발동하지 않는다) |

## 관리 규칙

- 규칙 추가 전 자문: "훅으로 강제 가능한가 / 가끔 쓰는 절차인가 / 매 턴 참인 상수인가"
- 새 규칙은 그 실패가 실재하고 산문으로 막히는 유형인지 확인한 뒤에만 추가한다.
- 슬림 유지: 새 규칙을 넣을 때 대응하는 옛 규칙·중복을 함께 정리한다.
