# 02. CLAUDE.md 구성 방식

전역 규칙 파일. **84줄 / ~2.1k 토큰** (권장 5k 이하). 매 세션 로딩되므로 원칙만 담고 상세는 링크로 위임

## 핵심설계방식

| #   | 원칙                                      | 막는 실패              |
| --- | ----------------------------------------- | ---------------------- |
| 1   | 모르면 멈추고 묻는다 (가정·혼란 드러내기) | 가정하고 그냥 달림     |
| 2   | 요청한 것만, 최소로                       | 100줄이면 될 걸 부풀림 |
| 3   | 수정 전 읽고, 외과적으로                  | 안 시킨 것까지 건드림  |
| 4   | '됐다'는 증거로 (성공기준 세우고 검증)    | 검증 없이 "됐다"       |

## 관리 규칙

- 규칙을 추가하기 전에 자문: "이건 훅으로 강제할 수 있나(→훅), 가끔 쓰는 절차인가(→스킬), 매 턴 참인 원칙인가(→CLAUDE.md)"
- 슬림 유지: 새 규칙 넣을 때 대응하는 옛 규칙/중복을 함께 정리.
