# 06. Commands — 슬래시 커맨드

## 커맨드

| 커맨드             | 기능                                                  |
| ------------------ | ----------------------------------------------------- |
| `/brainstorm`      | 코드 전 가정 좁히고 접근법 2~3개 비교(Plan 모드 진입) |
| `/dev-docs`        | 전략 계획 + 작업 분해 문서 생성                       |
| `/dev-docs-update` | 대화 요약 직전 개발 문서 갱신                         |

## 운영 규칙

- 커맨드 추가 시 기존 커맨드·스킬과 중복 확인. 고유 가치가 있을 때만.
