# 03. Skills — 스킬 설명 + 트리거

## 트리거는 어떻게 작동하나

스킬은 각 `SKILL.md`의 **`description`을 Claude가 보고 자동으로 띄운다**. 별도 배선 파일 없음.

- 한국어로 자주 부르는 스킬(pdf 등)은 description에 한글 트리거 키워드를 박아둠 (예: pdf → PDF 파싱).
- 스킬이 적을수록 매칭 정확도↑ → 전역은 16개로 슬림 유지.

| census       | 전수 작업을 차집합으로 증명(분모 파일 고정→흔적→census_diff) | "전수", "전부", "모든", "M/N"          | 2026-07-16 |

> census는 2026-07-16 재설계의 핵심 스킬 — 전수 축소(조용한 샘플링)를 문장 규칙 대신 기계 절차로 막는다. 완료 계약의 `VERIFY:` 줄과 결합 ([08a](08a-completion-contract.md)).

## 운영 규칙

- 스킬 추가·삭제·이동 시 `skills/README.md` 목차를 같은 작업에서 갱신.
- 새 스킬을 전역에 두기 전 역할 겹침 확인 — 겹치면 전역에 두지 않는다.
