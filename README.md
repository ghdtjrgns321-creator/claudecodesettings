# claudecodesettings

Claude Code(`~/.claude`)의 전역 설정을 관리하는 저장소.

## 핵심 원칙

1. **컨텍스트 최소화 — 원칙은 CLAUDE.MD 로드, 절차는 필요시 호출**
   - CLAUDE.md에는 작업의 핵심 원칙만 5K 토큰 이하로 간결하게 작성
   - 구체적인 실행 절차는 스킬 및 플러그인 형태로 분리, 필요할 때만 로드하여 동작
2. **통제 수준의 분리 — 가이드는 스킬, 제약은 훅**
   - 상황에 따른 판단이 필요한 안내 사항은 스킬로 제공
   - 반드시 지켜야 하는 필수 규칙은 훅을 통해 시스템으로 강제 및 예외 차단
3. **작업 완료 검증 — 완료수치 제시 및 산출물 필수**
   - 단순 완료 선언을 배제, 전체 작업 대비 완료 비율(M/N)과 실제 대조 산출물을 검증
   - 요구된 증거가 불충분할 경우, 시스템(하네스 및 훅 게이트)이 작업 종료를 거부
4. **설정의 모듈화 — 전역설정, 포르젝트 설정 분리**
   - 전역 설정(`~/.claude`)에는 상황에 종속되지 않는 범용 환경과 안전장치만 배치
   - 특정 기술 스택 및 도메인 지식은 개별 프로젝트 환경 내에 독립적으로 구성
